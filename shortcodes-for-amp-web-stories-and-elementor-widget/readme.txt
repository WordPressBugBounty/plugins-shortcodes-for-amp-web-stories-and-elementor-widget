== Description ==

### Web Stories Widgets For Elementor

This plugin provides an Elementor widget for the **Web Stories** plugin, enabling you to seamlessly add Web Stories to your Elementor-designed pages.

> **Note:** This is an addon for the **[Web Stories](https://wordpress.org/plugins/web-stories/)** plugin. You must have the **[Web Stories](https://wordpress.org/plugins/web-stories/)** plugin installed to use this addon.

With this plugin, you can easily showcase recently created stories on any page. Use the following shortcode to display recent stories:

**Recent Stories Shortcode**  
<pre> [Recent-stories column="3" show-button="yes" show-no-of-story="all" button-text="Default view" order="DESC" btn-color="#8BCDA0" btn-text-color="#000"] </pre>

To display individual stories, use this shortcode:  
<pre>[webstory id="{add here story id}" button-text="Default view" show-button="yes"]</pre>

For complete documentation, visit the **[Shortcodes & Docs](https://docs.coolplugins.net/docs/shortcodes-for-amp-web-stories-and-elementor-widget/using-individual-shortcode/)** page.

You can also check out a quick video tutorial:  

https://www.youtube.com/watch?v=WjZvTdMtj8c

= Key Features =

* Elementor widget for displaying Web Stories.
* Button typography and style customization.
* Customizable button labels.
* Shortcode support for displaying individual stories.
* Shortcode support for displaying recent stories.
* Customizable recent story display options (number of stories, order, button text, colors).

This plugin is a third-party addon and is not developed by or affiliated with **Google** or **Web Stories** in any way.  
To use this plugin, you must have **[Web Stories](https://wordpress.org/plugins/web-stories/)** installed. Additionally, for Elementor integration, you can optionally install **[Elementor Page Builder](https://wordpress.org/plugins/elementor/)**.

== Frequently Asked Questions ==

= Do I need to install any other plugins for Web Stories Widgets For Elementor to work? =  
Yes, you need to have the **[Web Stories](https://wordpress.org/plugins/web-stories/)** plugin installed. If you want to use it with Elementor, you can also install the **[Elementor Page Builder](https://wordpress.org/plugins/elementor/)** plugin (optional but recommended).

= How can I display my Web Stories on Elementor pages? =  
After installing the plugin, you can add Web Stories to your Elementor pages using the Web Stories widget or by using the shortcodes provided by the plugin.

= Can I customize the display of Web Stories? =  
Yes, you can customize the number of stories, button text, button colors, and order of stories using the provided shortcodes or widget settings.

= How do I use the Recent Stories shortcode? =  
Use the following shortcode to display recent Web Stories on any page:  
[Recent-stories column="3" show-button="yes" show-no-of-story="all" button-text="Default view" order="DESC" btn-color="#8BCDA0" btn-text-color="#000"]

You can adjust the shortcode attributes to control how many stories appear, the column layout, button colors, and text.

= Can I display individual Web Stories on my site? =  
Yes, you can display individual Web Stories by using the following shortcode:  
[webstory id="{add here story id}" button-text="Default view" show-button="yes"] 
Just replace `{add here story id}` with the ID of the specific story you want to display.

= Does this plugin work with other page builders besides Elementor? =  
The main functionality of this plugin is optimized for Elementor through the widget, but you can still use the shortcodes to display Web Stories on other page builders or within the WordPress editor.

= Is this plugin affiliated with Google or the Web Stories plugin? =  
No, this is a third-party addon and is not affiliated with Google or the Web Stories plugin. However, it requires the **[Web Stories](https://wordpress.org/plugins/web-stories/)** plugin to function.

= Where can I find detailed documentation for using the shortcodes? =  
You can find full documentation on how to use the shortcodes and customize them here:  
[Shortcodes & Docs](https://docs.coolplugins.net/docs/shortcodes-for-amp-web-stories-and-elementor-widget/using-individual-shortcode/)

== Installation ==

1. Install **Web Stories Widgets For Elementor** from the WordPress.org repository or by uploading the plugin's zip folder to the **/wp-content/plugins** directory.
   - Ensure you also have the **[Web Stories](https://wordpress.org/plugins/web-stories/)** plugin installed.
2. Activate the plugin through the **Plugins >> Installed Plugins** menu in WordPress.
3. After activation, you will find **Web Stories Widgets For Elementor** in the plugin section.
