== Description ==

### Web Stories Widgets for Elementor

**Web Stories Widgets for Elementor** is a simple and powerful addon that extends the official Web Stories plugin and adds a Web Stories widget inside Elementor.
With this plugin, you can easily display your Web Stories anywhere on your website using Elementor or shortcodes.

### Key Features

* **Elementor Widget :** Adds a dedicated Elementor widget so you can easily display Web Stories inside any Elementor. Just drag and drop the widget and choose your preferred story layout and options.

* **Shortcode Friendly:** Whether you are working with Elementor, Gutenberg, or Classic Editor, simply paste the shortcode and display your web stories.

* **Recent Stories Display:** Show your most recently created Web Stories anywhere on your website using a shortcode or the Elementor widget.
 
Use the shortcode below and replace the ID with your story's ID:

**Read Full Documentation:-** [Shortcodes & Docs](https://docs.coolplugins.net/docs/shortcodes-for-amp-web-stories-and-elementor-widget/using-individual-shortcode/)


[webstory id="{ add here story id}" button-text="Default view" show-button="yes"]

* **Multiple Stories:** You can display all stories or limit the number of stories to any value you prefer.

* **Choose Story Order:** Sort your Web Stories in ascending (ASC) or descending (DESC) order. 

* **Typography and Style Settings:** Customize button fonts, size, spacing, and color directly from Elementor. This makes it easy to match your theme's style.

== Frequently Asked Questions ==

= How do I display a Web Story in Elementor? =

You can use the dedicated Web Stories Elementor widget. Just drag and drop the widget onto your page and configure your story options from the widget settings.

= Can I display a single Web Story using a shortcode? =

Yes. Use the shortcode below and replace the {story-id} with your actual story ID:
[webstory id="{story-id}" button-text="Default view" show-button="yes"]

= Where can I find the story ID for the single story shortcode? =

When editing any Web Story, the story ID appears in the browser URL as post=123. Use that number as your story ID.

= Why are my stories not showing? =
Make sure that:
- The Web Stories plugin is active.
- You have published at least one story.
- You are using the correct shortcode or widget settings.
